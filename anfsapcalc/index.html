Azure NetApp Files for SAP Cost Estimator

Welcome!

IMPORTANT: This tool is designed to estimate the infrastructure costs of an SAP HANA on Azure NetApp Files landscape. However, it is not meant to be a comprehensive tool and may not account for all possible factors that may impact the total cost. Especially the backup cost is based on user provided assumed change rates. Therefore, it should only be used as a guide for estimating total costs.

Help

Add SAP HANA Systems and Volumes to compute the TCO for Azure NetApp Files capacity pools and Azure NetApp Files backup required for the SAP landscape. Multiple systems and volumes can be entered. The Capacity Pool Summary and Volume Summary sections can be expanded to view the details of the data entered.

Use the Capacity Pool Region section to select a region for each capacity pool to estimate costs based on the desired region.

This calculator provides estimated costs for Azure NetApp Files capacity pools as shown in the Estimated ANF Storage Costs section as well as Azure NetApp Files backup costs as shown in the Estimated ANF Backup Costs section.

You can modify the settings/parameters to adjust the requirements to fit your particular use case or workload using the Performance Modifiers and Snapshot and Backup sections.

If you make a mistake, systems or volumes can be removed using the icon in the Volume Summary section.

Configurations can be exported and imported using the and buttons respectively, located in the top right corner of this page.
