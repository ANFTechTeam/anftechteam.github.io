Azure NetApp Files for SAP TCO Estimator

IMPORTANT: This tool is designed to help estimate the TCO of SAP HANA on Azure NetApp Files. It is not intended to be a comprehensive TCO calculator, but rather a tool to help estimate the TCO of SAP HANA on Azure NetApp Files.

Help

Add SAP HANA Systems and Volumes to compute the TCO for Azure NetApp Files capacity pools and Azure NetApp Files backup. Multiple systems and volumes can be entered. The Pool Group Summary and Volume Summary sections can be expanded to view the details of the data entered.

This TCO calculator provides estimated costs for Azure NetApp Files capacity pools as shown in the Estimated Volume Costs section as well as Azure NetApp Files backup costs as shown in the Estimated Backup Costs section.

You can modify the settings/parameters to adjust the requirements to fit your particular use case or workload using the Performance Modifiers and Snapshot and Backup sections.

If you make a mistake, systems or volumes can be removed using the icon in the Volume Summary section.

Configurations can be exported and imported using the and buttons respectively, located in the top right corner of this page.
